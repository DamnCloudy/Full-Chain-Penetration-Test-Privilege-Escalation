# Full-Chain-Penetration-Test-Privilege-Escalation
This repository documents a black-box penetration test of a CTF system, covering reconnaissance, exploitation, and privilege escalation to root in a controlled lab environment.

Full Report (PDF): [View / Download](Full-Chain%20Penetration%20Test%20%26%20Privilege%20Escalation.pdf)

Summary: 
The assessment identified critical vulnerabilities including SQL injection, directory listing, exposed services, unrestricted file upload leading to RCE, and privilege escalation via misconfigured cron jobs and sudo permissions. These issues enabled full system compromise from initial access to root.

Tools
- Nmap
- Gobuster
- Burp Suite
- SQLMap
- Netcat
- Kali Linux

⚠️ Notice

All testing was conducted in a controlled, isolated lab environment for educational purposes only.
